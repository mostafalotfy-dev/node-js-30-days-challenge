# Node.js 30 Days Challenge 🚀

A comprehensive 30-day journey to mastering Node.js, from core concepts to building full-scale web applications.

## 📅 Roadmap

### Week 1: Node.js Basics & Express
- [x] **Day 1**: Initializing the project and understanding the runtime.
- [x] **Day 2**: Node.js core modules (HTTP, FS) and handling basic server logic.
- [x] **Day 3**: Introduction to Express.js (Middleware, Routing, Static Files, and HTML).
- [ ] **Day 4**: Template Engines (EJS/Pug).
- [ ] **Day 5**: The Model View Controller (MVC) pattern.

---

## 🛠️ Tech Stack
- **Runtime**: Node.js
- **Framework**: Express.js
- **Tools**: Nodemon, Body-Parser, etc.

## 📖 How to Run
1. Navigate to the specific day's folder:
   ```bash
   cd "Day X"
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```
