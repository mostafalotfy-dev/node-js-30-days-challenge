- [x] **[Day 1](./Day%201)**: Initializing the project and understanding the runtime.

## 🛠️ Tech Stack
- **Runtime**: Node.js
- **Framework**: Express.js
- **Templating Engines**: EJS, Pug, Handlebars
- **Tools**: Nodemon, Body-Parser, etc.

## 📖 How to Run
1. Navigate to the specific day's folder:
   ```bash
   cd "Day X"
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```
